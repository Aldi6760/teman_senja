# ☕ Teman Senja - Coffee Shop Website

**Teman Senja** adalah proyek landing page website untuk Coffee Shop yang dirancang dengan estetika modern, minimalis, dan elegan. Website ini bertujuan untuk memberikan pengalaman visual yang menenangkan bagi pengunjung, mencerminkan suasana hangat menikmati kopi di waktu senja.

## 🚀 Fitur Utama
- **Responsive Design**: Tampilan yang optimal di berbagai perangkat (Desktop, Tablet, & Mobile) menggunakan Bootstrap 5.
- **Hero Section**: Header yang menarik dengan tipografi elegan dan pesan selamat datang yang hangat.
- **Story Section**: Bagian "Our Story" yang menceritakan filosofi dan dedikasi Teman Senja.
- **Interactive Menu**: Katalog produk yang rapi untuk kategori **Coffee** dan **Dessert**.
- **Dynamic Product Modal**: Detail informasi produk yang muncul secara interaktif saat tombol dipesan diklik.
- **Image Gallery Slider**: Carousel yang menampilkan suasana *cozy* dari kafe.
- **Contact & Location**: Formulir kontak terintegrasi dan integrasi Google Maps untuk memudahkan pelanggan berkunjung.

## 🛠️ Teknologi yang Digunakan
- **HTML5**: Struktur konten web.
- **CSS3**: Styling kustom dengan variabel warna dan micro-animations.
- **Bootstrap 5**: Framework utama untuk tata letak responsif dan komponen UI.
- **JavaScript**: Logika untuk manipulasi modal produk dan interaksi lainnya.
- **Bootstrap Icons**: Library ikon yang konsisten.
- **Google Fonts**: Menggunakan kombinasi font *Dancing Script*, *Playfair Display*, dan *Inter*.

## 📂 Struktur Proyek
```text
Teman-Senja/
├── Asset/          # Koleksi gambar (kopi, dessert, slider, dll)
├── index.html      # File utama website
├── style.css       # Kustomisasi desain dan tema warna

```

## 💻 Cara Menjalankan Proyek
1. Clone repositori ini:
   ```bash
   git clone https://github.com/username/teman-senja.git
   ```
2. Masuk ke direktori proyek:
   ```bash
   cd teman-senja
   ```
3. Buka file `index.html` menggunakan browser favorit Anda.
